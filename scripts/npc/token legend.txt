CD - cooldown
MP - mana cost
HP - health cost
AS - base attack time
MS - base move speed
TR - turn rate
CR - critical strike damage
CC - disable debuff duration
MR - mana regeneration
__S - suffix for S class (unreduced)
__A - suffix for A class (normal)
__B - suffix for B class (moderate handicap)
